# YouTube 자막 추출 및 요약

YouTube 영상 URL을 받아 자막을 추출하고 한국어로 요약합니다.

## 사용법

```
/youtube-summarize <youtube-url>
```

## 실행 단계

### 1. 영상 메타데이터 확인

```bash
mmk youtube metadata <youtube-url>
```

영상 제목, 채널명, 게시일 등을 확인합니다.

### 2. 자막 추출

```bash
mmk youtube transcript <youtube-url>
```

자막이 없는 경우 사용자에게 알립니다.

### 3. 요약 생성

추출된 자막을 분석하여 다음 형식으로 한국어 요약을 작성합니다:

```
## 핵심 요약

**제목**: {영상 제목}
**채널**: {채널명}
**게시일**: {게시일}

### 주요 내용
• {핵심 포인트 1}
• {핵심 포인트 2}
• {핵심 포인트 3}
• {핵심 포인트 4}
• {핵심 포인트 5}

### 시장 시사점
{증시/경제에 대한 시사점 1-2문장}
```

## 요약 가이드라인

- 핵심 포인트는 3~5개로 정리
- 각 포인트는 1-2문장으로 간결하게
- 숫자, 종목명, 지표 등 구체적 데이터를 포함
- 투자 관련 결론이나 전망이 있으면 반드시 포함
- 한국어로 작성
